# Filled governance records (Application section)

One governance record per dataset for the DSA 2026 paper (Problem 2). All five recent Africa-related datasets are included.

| File | Dataset | Domain | Geography |
|------|---------|--------|-----------|
| `harveststat_africa_governance.json` | HarvestStat Africa | Agriculture | Sub-Saharan Africa (33 countries) |
| `lsms_isa_governance.json` | LSMS-ISA harmonised | Agriculture | Ethiopia, Malawi, Mali, Niger, Nigeria, Tanzania, Uganda |
| `grow_africa_governance.json` | GROW-Africa | Agriculture | Africa-wide |
| `dhs_malawi_2024_governance.json` | Malawi DHS 2024 | Health | Malawi |
| `harvestnet_ethiopia_governance.json` | HarvestNet (Ethiopia) | Agriculture | Ethiopia (Tigray, Amhara) |

**Use in the paper:** Section 4 (Application). Describe each dataset briefly, show the relevant parts of the filled governance record (or a summary table), and explain how filtering by e.g. `permitted_use` or `consent_documented` would support ethical downstream ML or audit.
